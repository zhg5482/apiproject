# apiproject
    
#beego 安装
    cd $GOLANG/src
    go get github.com/astaxie/beego
#bee 安装
    cd $GOLANG/src
    go get github.com/beego/bee
    参考文档 ： https://www.cnblogs.com/zuxingyu/p/6016816.html
#创建项目
    bee api apiproject

    启动 ./main.go
    自动编译启动 bee run (修改后自动生效)
    参考文档 ： https://blog.csdn.net/loongshawn/article/details/54114606
